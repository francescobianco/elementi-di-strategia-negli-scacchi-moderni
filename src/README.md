# Sorgenti nuova edizione

- `main.tex`: ingresso della build LaTeX.
- `frontmatter/`: titolo, introduzione e materiali preliminari.
- `chapters/`: capitoli numerati del libro.
- `appendices/`: appendici in formato LaTeX omogeneo.
- `assets/diagrams/`: diagrammi del testo storico.
- `assets/appendix/media/`: diagrammi estratti dalla nuova appendice.

La notazione delle mosse usa gia' le figurine nei sorgenti (`¤`, `¥`, `£`, `¦`, `¢`), senza post-processing in build.
